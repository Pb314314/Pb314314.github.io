# Publication Preview Images

This directory contains preview images for publications displayed on the publications page.

## Required Images

Please add the following images to this directory:

1. **distca.png** - Preview image for DistCA paper (MLSys '26)
2. **seer.png** - Preview image for Seer paper (OSDI '26)
3. **kimi-linear.png** - Preview image for Kimi Linear tech report
4. **kimi-k25.png** - Preview image for Kimi K2.5 tech report

## Image Guidelines

- **Recommended size**: 200x200px to 400x400px
- **Format**: PNG or JPG
- **Aspect ratio**: Square (1:1) works best
- **File size**: Keep under 500KB for fast loading

The images will be automatically displayed next to each publication entry on the publications page.
