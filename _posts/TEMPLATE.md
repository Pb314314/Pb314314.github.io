## How to Write Blog Posts

1. Create a new file in the `_posts/` directory
2. Name it with the format: `YYYY-MM-DD-your-title.md`
3. Add the front matter (YAML header) at the top with:
   - `layout: post`
   - `title: Your Title`
   - `date: YYYY-MM-DD HH:MM:SS`
   - `description: Brief description`
   - `tags: [tag1, tag2]` (optional)
   - `categories: [category]` (optional)

4. Write your content below using Markdown

## Markdown Features

You can use:
- **Bold text**
- *Italic text*
- [Links](https://example.com)
- Code blocks with syntax highlighting
- Images
- Math equations (if enabled)
- And much more!

## Tips

- Use descriptive titles and descriptions
- Add relevant tags to help readers find your posts
- Include images in `assets/img/` and reference them with `![alt text](/assets/img/your-image.jpg)`
- Write about your research insights, thoughts, and learnings

Happy blogging!
